# Quickstart: 발제 중심 홈 피드 재정렬

## Preview

```bash
hugo server -D
```

Open `http://localhost:1313/` and verify:
- Judge card is centered and sorted newest-first.
- Black/White cards appear beneath the judge.
- Missing side renders a placeholder without breaking layout.

## Build Validation

```bash
hugo --minify
```
